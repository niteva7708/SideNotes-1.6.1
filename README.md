## What's New in SideNotes 1.6.1?

Version 1.6.1 is primarily a refinement update that improves stability and addresses user feedback following the major SideNotes 1.6 release.

### Disable Automatic Spaces When Pasting

One of the most requested refinements introduces an option to prevent automatic spaces from being inserted around pasted text.

This may sound minor, but users who frequently paste:

* URLs
* Code snippets
* Structured text
* Markdown content

will appreciate the improved control.

### Emoji Panel Improvements

The update fixes an issue where opening the emoji picker could unexpectedly close SideNotes if the "Close when clicking outside SideNotes" setting was enabled.

### Launch on Startup Fix

Users relying on automatic startup functionality will be pleased to know that Launch on Startup now works correctly again.

### Hot Side Improvements

A bug causing the Hot Side trigger area to remain visible while selecting text has been resolved.

### Bottom Bar Refresh Fix

The note count in the bottom bar now updates correctly.

### Font Crash Resolution

Version 1.6.1 also fixes a crash that could occur when selected fonts became unavailable.
